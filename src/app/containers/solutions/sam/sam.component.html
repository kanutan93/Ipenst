Что такое SAM?

Определение “SAM”: это набор бизнес-практик, включающих управление и оптимизацию закупок, внедрения, поддержки, использования и вывода из эксплуатации программных приложений внутри организации.

SAM – международный отраслевой стандарт. ISO/IEC 19770-1 поддерживается Microsoft и многими другими компаниями.

SAM проект поможет:

  • Разобраться в текущем использовании и лицензировании продуктов Майкрософт
  • Оптимизировать текущее использование ПО
  • Получить все выгоды корпоративного лицензирования
  • Обеспечить долгосрочное лицензионное соответствие
  • Получить информацию для принятия решения о развитии ИТ
  • Соответствовать Российским и мировым стандартам

Увеличение пользы от SAM

Каждый проект несет пользу. При проведении любого из проектов собирается большое количество данных, на основании которых могут быть сделаны ценные выводы. Используя эти данные, есть возможность понять ключевые задачи и выявить области для совместного бизнеса.

Как выглядит проект SAM

Ход каждого проекта разбивается на этапы. Выполнение этапов в данной последовательности и в полном объеме в соответствии с требованиями программы позволяет сделать заключительные документы максимально полезными для клиента

  • На стадии планирования производится сбор данных об инфраструктуре клиента, текущих задачах и будущих планах. Партнер определяет список проектной группы, способы сбора данных, сроки и рамки проекта.
  • Осуществляется сбор данных об установленном ПО и имеющихся лицензиях. Может потребоваться несколько способов для сбора всех необходимых данных и получения достоверной картины, включая интервью сотрудников и анализ внутренних документов.
  • Анализ данных предусматривает проверку на достоверность всех полученных данных об установленном и используемом ПО и имеющихся лицензиях. Производится оценка текущих подходов и политик SAM и готовятся рекомендации по улучшению в соответствии с ожиданиями клиента
  • При окончании проекта результаты, рекомендации и предложения по дальнейшим действиям представляются клиенту в кратком отчете и наборе более детальных документов.
